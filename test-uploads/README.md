# 📂 Pasta de Testes

Use esta pasta para fazer upload de arquivos e testar funcionalidades do Gerenciador de Mods.

## Como usar
1. Faça upload dos seus arquivos de mod aqui (`.package`, `.ts4script`, etc.)
2. Informe ao assistente quais funcionalidades deseja testar
3. Os testes serão realizados com base nos arquivos enviados

## Tipos de arquivos suportados pelo jogo
- `.package` — Mods e CCs (Custom Content)
- `.ts4script` — Mods com scripts
